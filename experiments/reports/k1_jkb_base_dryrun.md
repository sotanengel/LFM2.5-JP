## 系全体としての解釈

1. **base = JP-tuned なだけあり core レベル知識は 64% と実用ライン**。だが standard で 50%、advanced で 36% に落ちる勾配は明確 → 「概ね担保」の目標(親 Epic #120)には **standard の底上げが最優先**。
2. **分野別で明確な二極化**:
   - **強い**(65%+): 宗教・伝統芸能・地理基本 = 古代日本の中心テーマ、Wikipedia-ja の教科書的記述が濃い領域
   - **弱い**(< 45%): 生活・慣習、言語、食文化(郷土)、科技産業 = **現代の実務知識と地方文化**、いずれも Wikipedia-ja でも横断的に薄い / 記事は存在するが個別記事に散らばる
3. **advanced が 36% と想定より高い理由**は MCQ の 25% 下駄が乗る効果と、matrix 選定で「専門」と「マイナー」を混同した可能性の混合。**K1 本評価で MCQ 除外スコアを追記**して分離する。

## K1 目標水準の暫定固定(実測を見て提案、要ユーザー確認)

| 難度 | 実測(base JP-202606) | 提案目標水準(K2 cpt-D 達成時) |
|---|---|---|
| core | 64.1% | **≥ 75%**(+11pt。JP-tuned の残る core ギャップを埋める) |
| standard | 49.6% | **≥ 65%**(+15pt。**最重要**。中学生水準の日本知識をカバー) |
| advanced | 36.3% | **≥ 45%**(+9pt。専門知識は 1.2B の限界に近いので底上げ小さく) |
| 全体 | 49.9% | **≥ 60%**(+10pt) |

**判定規則(K1 セッション冒頭で確定)**: cpt-D が JKB **全体 +8pt 以上 かつ** 全 12 分野で -3pt 以内(局所劣化なし)かつ IFEval prompt_strict -3pt 以内 かつ llm-jp-eval AVG -1pt 以内 → 採用。

## K2 (Issue #123) CPT ターゲット選定(次アクション)

上記ワースト 10 セルの直接的な対応:

- **生活・慣習(3 セル)**: 祝日カテゴリ、行政手続き、社会保険、冠婚葬祭系 Wikipedia-ja 記事の重点選抜。政府の白書等オープンデータも候補。
- **言語(1 セル)**: `Category:日本語学`、`Category:日本語の方言`、古典文法系記事の重点選抜。
- **食文化(2 セル)**: `Category:郷土料理`、`Category:日本の伝統食` の重点選抜。
- **科学技術・産業 standard(1 セル)**: `Category:日本のノーベル賞受賞者` + 各受賞者ページ、企業カテゴリ。
- **歴史 advanced / 政治・制度 advanced(2 セル)**: `Category:日本の憲法` + 戦前事件記事。ここは 1.2B の限界に近い可能性が高いので優先度低。
